# Commands

Repositorio de documentación — no hay comandos de build, test o despliegue automatizados.

| Propósito | Acción |
|-----------|--------|
| Editar propuesta | Editar `docs/propuesta-arquitectura-hurto-vehiculos.md` |
| Ver caso de estudio | Abrir `docs/Caso de Estudio1 - DETECCION VEHÍCULOS HURTADOS.pdf` |
| Vista previa diagramas Mermaid | VS Code + extensión Mermaid Preview / mermaid.live |
| Lint Markdown (opcional) | `npx markdownlint-cli "docs/**/*.md"` |
